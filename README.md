# Ruboty::Tryruby

Ruboty plugin to evaluate one line of Ruby script

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ruboty-tryruby'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ruboty-tryruby
